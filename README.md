# 🏍️ TVS Bike Sales Website

A complete responsive bike sales website with user authentication, product catalog, and order management system.

## 🌐 Live Demo
[View Website](https://akshayg16.github.io/Bike_Sale/)

## 📱 Features
- ✅ **User Authentication** - Sign up / Login (saves data locally)
- ✅ **Product Catalog** - Motorcycles, Scooters, Electric Vehicles
- ✅ **Order Management** - Track your purchases
- ✅ **Payment Demo** - Card, UPI, Net Banking options
- ✅ **Services Page** - Maintenance, customization, assistance
- ✅ **About Us** - Company story and values
- ✅ **Fully Responsive** - Works on all devices

## 🚀 Technologies Used
- HTML5
- CSS3
- JavaScript
- LocalStorage (for data persistence)
